Python Type Checker (AST-based)
Un outil léger pour vérifier la présence d'annotations de type (type hints) dans les fichiers Python au sein d'une structure d'exercices (ex0, ex1, etc.).

Fonctionnalités
Vérifie les arguments de fonctions.

Vérifie les valeurs de retour.

Calcule un score de couverture de typage.

Supporte les sorties en couleur dans le terminal.

Utilisation
python3 typing_check.py [--verbose] [--ex 0 1 2]
